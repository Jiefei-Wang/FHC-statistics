Title: A novel exact equal local levels higher criticism test
Paper authors: Jiefei Wang, Jeffrey C. Miecznikowski
Code author: Jiefei Wang (jwang96@buffalo.edu)
Software:
MATLAB:
9.0.0.341360 (R2016a)
February 11, 2016


Description:
This code computes the critical value of The FHC statistic and the P-values for an observed FHC statistic. The algorithm is from "A novel exact method for significance of higher criticism via Stecks determinant(Jeffrey C. Miecznikowsk)" with slight changes.

The file "main_critical.m" demonstrate the usage of each function
