Title: A novel exact equal local levels higher criticism test
Paper authors: Jiefei Wang, Jeffrey C. Miecznikowski
Code author: Jiefei Wang (jwang96@buffalo.edu)
Software:
MATLAB:
9.0.0.341360 (R2016a)

Description:
This code computes the exact power of HC and FHC statistic for a beta-uniform mixture distribution.

The file "main_power.m" computes the power for different sample size, alpha,beta parameters in beta distribution
